# LatihasChocobo（TC版）

自動陸行鳥競賽 Dalamud 插件，針對 FFXIVSimpleLauncher（TC）環境調整。

## 使用方式

- 輸入 `/lc` 或 `/latihaschocobo` 開啟插件介面

## 功能

### 競賽自動化
- 自動加速、轉向、跳躍
- 自動使用道具（技能1），偵測欄位有道具時立即使用
- 自動使用技能2（體力 > 70% 或進度 > 75% 時，最短冷卻20秒）
- 低體力/路長時禁用超速加速；高體力/路長時強制加速（可設定）
- 競賽結束後自動點擊結果確認視窗
- 進入指定地點自動循環匹配（任務搜尋器延遲3秒點擊，避免載入未完成）
- **陸行鳥升至40級時自動停止循環並發出提示音**

### 障礙物偵測與迴避
- **紅陷阱（2005040）**：正前方近距離時跳躍，其他情況向左閃避
- **紫減速（2005039）**：一律向左閃避
- **障礙怪物（3595 靜止 / 3596 移動）**：閃避
- **特殊NPC（3597）**：主動靠近

### 寶箱/加速偵測
- **藍加速（2005038）**、**黃/藍寶箱（2005024/2005025）**、**綠體力（2005041）**：主動靠近或跳躍拾取
- 壞物件優先於好物件處理，避免因靠近好物件而撞上障礙

### 資訊顯示
- 當前等級、經驗值（含上場獲得）、剩餘訓練次數
- 各屬性培育百分比（最高速度/加速力/體力/持久力/適應力）
- 先天/後天技能名稱
- 是否超速、是否可用道具、加速狀態
- 背包配種/出賽登記書星級分析
- 物件偵測頁（顯示附近所有物件 DataId，用於識別新障礙）

## TC 相關調整
- 材質路徑不含 `_hr1` 後綴（例如 `070101.tex`、`tc/180043.tex`）
- 已修正道具可用判斷、超速偵測材質路徑

## 需要手動設定的部分

### 1. 按鍵綁定（必要）
插件透過模擬按鍵操作遊戲，**必須確認遊戲內按鍵設定與插件「按鍵」頁面的 VK code 一致**。
- 預設假設 W=前進、A=左、D=右、Space=跳躍、1=技能1、2=技能2
- 若遊戲按鍵不同，在「按鍵」頁面填入正確的 [Windows VK code](https://learn.microsoft.com/en-us/windows/win32/inputdev/virtual-key-codes)

### 2. 技能欄位配置（必要）
- 技能1（KC_1）：賽道道具格，需對應到會使用當前持有道具的按鍵
- 技能2（KC_2）：建議對應高速衝刺或體力相關技能

### 3. 任務搜尋器選擇任務（使用自動匹配功能時）
插件只能自動**開啟任務搜尋器並點擊加入按鈕**，**無法自動選擇要報名的競賽項目**。
- 首次使用前，請先手動開啟任務搜尋器，選好陸行鳥競賽項目
- 之後插件的「立刻匹配」或自動循環才能直接點加入

### 4. 自動循環區域（使用自動匹配功能時）
「循環區域」欄位需填入**離開競賽後會回到的地點 TerritoryType ID**（用 `|` 分隔多個）。
- 可在插件主介面「當前區域」查看目前的 TerritoryType
- 競賽場地本身（389/390/391）不需填入，插件會自動偵測

### 5. 遊戲視窗
插件透過 `SendMessage` 傳送按鍵，**遊戲視窗不需要在最前面**，但必須保持開啟（不能最小化）。

### 6. 「滿級模式」選項
若陸行鳥已滿級，勾選「滿級模式」可放寬障礙迴避距離判斷（從14→20、22→30），更積極地接近好物件。
